# API Test Automation framework challenge
## Andres Sebastian Cespedes Cubides

A test automation framework for testing the TheMovieDB API in 3 features: Authentication, Lists and Movie rating

## Tools used:
- Java JDK 1.8
- Maven
- Serenity BDD
- Cucumber
- REST Assured
